# Code of Conduct

## Our commitment

We expect contributors to act respectfully and professionally.

## Unacceptable behavior

- Harassment or discriminatory language
- Publishing secrets or private data
- Deliberately destructive changes

## Enforcement

Project maintainers may remove content, close issues, or reject contributions that violate this policy.

## Scope

This code of conduct applies to all project spaces, including issues, pull requests, and discussions.
